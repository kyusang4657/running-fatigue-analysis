# Running Fatigue Analysis

## 프로젝트 개요

본 프로젝트는 스마트폰 IMU 센서와 러닝 로그를 활용하여 개인 기준선 기반 러닝 피로 변화를 분석하는 연구 프로젝트입니다.

스마트폰 IMU 데이터는 본 연구의 핵심 데이터로 사용되며, Galaxy Fit3와 Samsung Health에서 확인할 수 있는 심박수, 수면 시간, 운동 기록 등은 선택적인 보조 데이터로 활용합니다.

기존 상용 러닝 앱이 거리, 페이스, 심박수 중심의 기록과 코칭을 제공한다면, 본 프로젝트는 러닝 초반 구간을 개인 기준선으로 설정하고 후반부의 센서 변화량을 분석하여 피로 변화의 판단 근거를 설명하는 것을 목표로 합니다.

본 프로젝트는 삼성 헬스와 같은 상용 앱을 대체하는 것이 아니라, 상용 앱의 운동 기록을 보완하는 설명 가능한 피로 변화 분석 시스템을 목표로 합니다.

---

## 연구 주제

**개인 기준선 기반 스마트폰 IMU 러닝 피로 변화 지수 설계 및 설명 가능한 피드백 시스템**

---

## 연구 배경

삼성 헬스, Garmin, Strava와 같은 상용 러닝 앱은 거리, 시간, 페이스, 심박수, 칼로리 등 다양한 운동 기록을 제공한다. 그러나 사용자가 러닝 후반부에 피로를 느꼈을 때, 그 피로가 어떤 센서 변화에서 비롯되었는지 설명하는 데에는 한계가 있다.

본 프로젝트는 상용 러닝 앱을 대체하는 것이 아니라, 상용 앱의 거리·페이스 중심 기록을 보완하는 연구용 피로 변화 분석 시스템을 목표로 한다.

---

## 핵심 아이디어

러닝 초반 구간을 개인 기준선으로 설정하고, 후반 구간에서 다음과 같은 변화가 나타나는지 분석한다.

* 페이스 저하율
* 상하 흔들림 증가율
* 좌우 흔들림 증가율
* 움직임 리듬 불안정성 증가율

이러한 변화량을 바탕으로 RFCI, 즉 Running Fatigue Change Index를 계산하고, 피로 변화 판단 근거를 사용자에게 설명한다.

| 구분     | 상용 러닝 앱          | 본 프로젝트                  |
| ------ | ---------------- | ----------------------- |
| 목적     | 운동 기록 관리, 코칭 제공  | 피로 변화 분석 방식의 실험과 검증     |
| 핵심 데이터 | 거리, 시간, 페이스, 심박수 | 스마트폰 IMU, 러닝 로그, 개인 기준선 |
| 분석 방식  | 상용 알고리즘 기반       | 개인 기준선 대비 변화량 분석        |
| 결과 표현  | 운동 기록 요약, 훈련 계획  | 피로 변화 지수와 판단 근거 제공      |
| 목표     | 사용자 서비스          | 연구 및 실험 검증              |

## 주요 기능

* 스마트폰 IMU 센서 데이터 전처리
* 러닝 초반/중반/후반 구간 분리
* 개인 기준선 대비 변화량 feature 추출
* RFCI, Running Fatigue Change Index 계산
* 피로 변화 판단 근거 출력
* Streamlit 기반 분석 대시보드 구현
* 규칙 기반 지수와 머신러닝 모델 비교
